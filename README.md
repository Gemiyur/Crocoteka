# Крокотека
Библиотека электронных книг на локальном компьютере.

Стандартно в библиотеку могут быть добавлены:
- Аудио книги: M4B, MP3
- Текстовые книги: EPUB, FB2, FB3, PDF, TXT

Расширения файлов книг могут быть добавлены или удалены в настройках.  
Библиотека поддерживает серии и жанры (теги) книг, сведения об авторах и комментарии к файлам книг.

Запущен может быть только один экземпляра приложения.  
В случае попытки запуска, при запущенном приложении, активируется уже работающий экземпляр.
