# Крокотека
Библиотека электронных книг.

Поддерживаются электронные книги в формате FB2 и аудиокниги в форматах M4B/M4A/MP3.  
Аудиокнига должна состоять только из одного файла.  
Если файлов несколько, то объединять их можно через цикл (серию) книг.

Идея обкатывается в проекте Говорун (Govorun).

## Технологии
- Интерфейс: WPF
- Платформа: NET 8.0
- База данных: LiteDB
