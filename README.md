# Крокотека
Библиотека электронных книг.

Каталог аудио и текстовых книг на локальном компьютере.  
Стандартно в библиотеку могут быть добавлены аудио книги (M4B и MP3) и текстовые книги (FB2 и EPUB).  
Расширения файлов книг могут быть добавлены или удалены в настройках.  
Библиотека поддерживает серии книг, сведения об авторах и жанры (теги) книг.
